Il faut qu'un admin puisse gérer des produits sur un site de vente : - ajouter des produits - afficher tous les produits - supprimer un produit - Afficher et supprimer : sur la même page - un tableau avec tous les produits - dans chaque ligne représentant un produit, un bouton/lien pour supprimer le produit Respecter le MVC, créer les classes d'objet correspondantes, et adapter les templates html Les produits doivent avoir ces propriétés : - id - createdAt - name - price (int) - description - isOnline (boolean) Il faut donc créer - la base de données correspondante - les models, ou entities, correspondants - le formulaire correspondant Le controller index.php doit faire les traitements et afficher les produits A vous de faire - un autre controller pour les traitements liés à l'enregistrement d'un produit et l'affichage du formulaire - un autre controller pour supprimer un produit
